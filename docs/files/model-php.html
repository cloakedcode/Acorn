query

static function query($class,
$query)

Executes a (prepared) query and returns an array of models.  Substitutes ‘#table’ in the query with the name of the table.  Any arguments beyond $query are substituted into the query.

<code> <?php $user = User::query(‘SELECT * FROM #table WHERE id=?’, 10); ?> </code>

@param string $class Class of the model.  @param string $query SQL query.  @static @access public @return bool|array False if query was not successfully executed, otherwise models from query.
